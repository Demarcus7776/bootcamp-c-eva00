# 🚀 42 C Trainer

A modern web app to test and improve your C programming skills, inspired by the 42 school evaluation system.

---

## 🧠 Features

* 🧪 **Quiz System**

  * 20+ C questions (basics → advanced)
  * Random 10 questions each session

* 💻 **Coding Challenges**

  * 42-style problems (ft_strlen, argv, etc.)
  * 10 random challenges per run
  * Total score out of 100

* 🏆 **Rating System**

  * oh hellll nah twin 💀
  * maybe 😐
  * u r the goat 🐐🔥

* 🧠 **Smart Coaching**

  * Personalized feedback based on your score
  * Roadmap to improve your C skills

* 🔁 **Infinite Practice**

  * Train Again button
  * New questions & challenges every time
